Docker

  • Open platform to build, ship, and run applciations
  • Immutable infrastructure
  • Automation through Dockerfiles
  • Virtual environment through containerization

What is con... contai... that?

  • Containerization uses Copy-on-write file system layers to build a new system
  • Immutablility at the system level
  • Isolation and Encapsulation

Dockerfile

  • Start with a base image
  • Define a series of modifications
  • Simple syntax to define a system
  • Store system definition in code repository

Sample Dockerfile


FROM ubuntu:14.04
MAINTAINER Bill Carlson <bill@coacoas.net>

RUN apt-get update -y
RUN apt-get install -y curl nginx
RUN update-rc.d -f nginx remove && \
    echo "\ndaemon off;" >> /etc/nginx/nginx.conf

EXPOSE 80
EXPOSE 443

CMD ["nginx"]
